## 🔧 技术配置

### 前端配置
`frontend-cloud.html` 中已预设：
```javascript
const CLOUD_SERVER = {
    ip: '43.128.253.193',
    port: '3001',
    protocol: 'http'
};
```

### 网络要求
- **互联网连接**：需要稳定的网络连接
- **端口开放**：确保本地防火墙允许出站连接
- **DNS解析**：能够解析域名和IP

### 安全连接
- **HTTP协议**：目前使用HTTP（可升级HTTPS）
- **数据加密**：敏感数据本地加密后传输
- **访问控制**：基于导演身份验证

## 🛠️ 故障排除

### 连接失败
1. **检查网络**：确保互联网连接正常
2. **验证IP**：确认服务器IP正确
3. **测试端口**：使用 `telnet 43.128.253.193 3001` 测试连接
4. **检查防火墙**：确保本地防火墙未阻止连接

### 连接测试命令
```bash
# 测试云端API连接
curl http://43.128.253.193:3001/api/directors/me

# 测试服务器状态
curl http://43.128.253.193:3001

# 测试网络延迟
ping 43.128.253.193
```

### 常见问题
1. **连接超时**
   - 原因：网络延迟或服务器负载
   - 解决：稍后重试或检查本地网络

2. **API错误**
   - 原因：服务器维护或更新
   - 解决：等待服务器恢复或联系管理员

3. **数据不同步**
   - 原因：网络中断导致同步失败
   - 解决：重新连接或手动刷新
